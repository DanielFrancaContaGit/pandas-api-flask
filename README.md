# pandas-api-flask


### How to start the project

- Create a virtual environment

```
python3 -m venv venv
```

- Activate the virtual environment

```
source venv/bin/activate
```

On Windows:

```
venv\Scripts\activate
```

- Install the dependences

```
pip install requirements.txt
```

- Run the project

```
 python3 app.py
```
