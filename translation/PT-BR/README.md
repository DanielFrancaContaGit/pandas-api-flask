# pandas-api-flask


### Como iniciar o projeto

- Criar o ambiente virtual

```
python3 -m venv venv
```

- Ativar o ambiente virtual

```
source venv/bin/activate
```

No Windows:

```
venv\Scripts\activate
```

- Instalar as dependencias

```
pip install requirements.txt
```

- Rodar o projeto 

```
 python3 app.py
```
